RP-Hub 模板

                            

                                
默认作用域
global 是全局模板，导入 RP-Hub 时不需要先选择角色。

                                
                            

                            

                                
默认位置
模板在 RP-Hub UI 中的插入位置。

                                
                            

                            

                                
导出参数
默认启用才能参与 RP-Hub UI变量同步；排序默认 100；更新模式默认 merge。

                                

                                    
                                    
                                    
                                

                            

                        


                        

                            
AI 服务

                            

                                
API 地址
可填根域名、/v1 或完整 /v1/chat/completions，工具会自动补全。

                                

                                    
                                    
未获取模型列表。

                                

                            

                            

                                
API 密钥
只保存在当前浏览器的 localStorage。本地文件直接打开时，接口仍可能受 CORS 限制。

                                

                                    
                                    

                                

                            

                            

                                
模型
从接口返回的模型列表中选择，不使用浏览器原生下拉。

                                

                                    
                                    
                                    

                                

                            

                            

                                
流式输出
开启后按 OpenAI SSE 格式实时显示 AI 生成内容；关闭后一次性显示结果。

                                
                            

                            

                                
最大输出 Token
AI 单次输出的最大 token 数。复杂 UI 模板需要较大值（建议 8192-16384）。设置过低会导致输出被截断、模板不完整。
